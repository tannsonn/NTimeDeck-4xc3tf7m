# NTimeDeck

NTimeDeck は、タイマー・アラーム・ポモドーロ機能を備えたWebアプリケーションです。

## 📌 概要 (Overview)
日々のタスク管理や学習の生産性を高めるために、3つの主要機能（タイマー・アラーム・ポモドーロ）を1つの洗練されたUIに統合しました。グラスモーフィズム（ガラス風）デザインを採用し、Notionのダークモード環境に美しく馴染むように設計されています。

## ✨ 主な機能と特徴 (Features)

### 1. タイマー機能 (Timer)
* 複数のタイマープリセットを保存・管理可能
* 横スクロール（カルーセル）UIによる直感的な切り替え
* よく使うタイマーのピン留め機能

### 2. アラーム機能 (Alarm)
* **曜日指定（繰り返し）機能**: 日〜土の任意の曜日を選択して繰り返し発火が可能
* **スヌーズ機能**: 指定した間隔（分）でのスヌーズに対応
* スマートなカードUI: 選択された曜日だけをコンパクトにインジケーター表示

### 3. ポモドーロ機能 (Pomodoro)
* 作業（Pomodoro）、短い休憩、長い休憩をサポート
* **自動進行（Auto-Advance）**: 作業→短い休憩を4サイクル繰り返した後、自動的に長い休憩へ移行する設定が可能

### 4. 高度なUI/UXとアクセシビリティ (UI/UX Highlights)
* **ダークフィルター**: どんな背景画像を設定しても文字やボタンの視認性が落ちないよう、背景とUIの間に最適な暗さのレイヤーを配置
* **キーボードショートカット**: 設定画面や追加モーダルにおいて、`Enter`キーで保存、`Esc`キーでキャンセルの操作に対応
* **誤操作防止**: モーダル内でテキストをドラッグ選択したまま領域外でマウスを離しても、意図せず画面が閉じてしまわないように厳密なクリック判定を実装

## 🛠 技術スタック (Tech Stack)
* **フロントエンド**: HTML5, CSS3, Vanilla JavaScript (フレームワーク不使用)
* **アイコン**: Lucide Icons
