# aesthetic-timer

Aesthetic Timer は、Notionなどへの埋め込み（Embed）を意識してデザインされた、美しく多機能なタイマー・アラーム・ポモドーロWebアプリケーションです。洗練されたUIと直感的な操作性、そして日々の作業を快適にするための細やかなUXを備えています。
外部ライブラリやフレームワークに依存せず、HTML / CSS / Vanilla JavaScript のみで軽量に構築されています。

## ✨ 主な機能

### 1. タイマー機能
- 複数のタイマーを保存・管理。
- ワンクリックでの切り替えや、よく使うタイマーの「ピン留め」機能。
- 残り時間が一目でわかる巨大な中央タイマーディスプレイ。

### 2. ポモドーロ・テクニック
- 集中（Pomodoro） / 短い休憩（Short Break） / 長い休憩（Long Break）をサポート。
- 各セッションの時間を自由にカスタマイズ可能。
- 「自動進行（Auto-Advance）」をONにすることで、セッション終了後に次のモードへ途切れることなく無限ループで自動スタートします。

### 3. アラーム機能
- 複数のアラームを設定・保存。
- **スヌーズ機能**: 指定間隔（分）で再度鳴動させることが可能。
- **繰り返し（曜日指定）**: 「月・水・金のみ」といった曜日ごとの繰り返し設定に対応。指定曜日以外は発火をスキップします。

### 4. 美しくカスタマイズ可能なUI
- **Glassmorphismデザイン**: 背景に溶け込む半透明なガラス風のUIパネル。
- **多彩な背景プリセット**: アプリの雰囲気をワンクリックで変更可能。
  - グラデーション（オーロラ、オーシャン）
  - 高画質写真（ネイチャー、宇宙・ギャラクシー、ミニマルデスク）
- どのような背景でも文字やボタンがクッキリ見えるよう、ダークレイヤーフィルターや視認性補正が施されています。
- **モバイルフレンドリー**: スマートフォンなどの小さな画面でもアラームの曜日インジケーター等がコンパクトに表示され、レイアウトが崩れません。

### 5. 音声カスタマイズ (MP3対応)
- 内部のビープ音ではなく、HTML5 `Audio` オブジェクトを活用。
- `./sounds/` ディレクトリに配置した任意の `.mp3` ファイルを読み込ませることで、タイマー、アラーム、ポモドーロそれぞれの鳴動音を自由にカスタマイズ可能です。

### 6. 洗練されたUX (操作感)
- **ショートカットキー対応**: モーダルを開いている際、`Enter` キーで保存、`Esc` キーでキャンセル（閉じる）が可能です。
- **ドラッグ誤操作防止**: モーダル内のテキストを選択中にうっかり外側でマウスを離しても閉じないよう、厳密なクリック判定を実装。
- **モーダル on モーダル**: 設定画面からアイテムを編集する際、設定画面が閉じずに上に重なるため、編集後もシームレスに元の画面に戻れます。

## 🚀 使い方

1. リポジトリのファイルをローカルにダウンロードします。
2. もしお好みの音源がある場合は、MP3ファイルを `sounds/` フォルダ内に配置し、`index.html` 内の `<select>` のパスを調整してください。
3. `index.html` をブラウザで開くだけで、すぐにすべての機能を利用可能です。
   （※ローカルストレージを利用して設定やタイマー状態を自動保存するため、次回起動時も状態が復元されます）

## 🛠 技術スタック

- **HTML5**
- **CSS3** (CSS Variables, Flexbox, CSS Grid, Glassmorphism)
- **Vanilla JavaScript** (ES6+)
- **Lucide Icons** (CDN経由でのアイコン描画)
- 状態管理: `localStorage`

## 📝 開発メモ
このアプリは「機能的でありながら、ずっと表示しておきたくなる美しいタイマー」をコンセプトに、UIの細部のマージンからアニメーションのパフォーマンス（描画遅延の解消など）に至るまで、徹底的に磨き上げられています。
