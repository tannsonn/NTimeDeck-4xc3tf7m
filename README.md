# NTimeDeck

NTimeDeck は、タイマー・アラーム・ポモドーロ機能を備えたWebアプリケーションです。

## 📌 概要 (Overview)
日々のタスク管理や学習の生産性を高めるために、3つの主要機能（タイマー・アラーム・ポモドーロ）を1つの洗練されたUIに統合しました。グラスモーフィズム（ガラス風）デザインを採用し、Notionのダークモード環境に美しく馴染むように設計されています。

## ✨ 主な機能と特徴 (Features)

### 1. タイマー機能 (Timer)
* 複数のタイマープリセットを保存・管理可能
* 横スクロール（カルーセル）UIによる直感的な切り替え
* よく使うタイマーのピン留め機能

### 2. アラーム機能 (Alarm)
* **曜日指定（繰り返し）機能**: 日〜土の任意の曜日を選択して繰り返し発火が可能
* **スヌーズ機能**: 指定した間隔（分）でのスヌーズに対応
* スマートなカードUI: 選択された曜日だけをコンパクトにインジケーター表示

### 3. ポモドーロ機能 (Pomodoro)
* 作業（Pomodoro）、短い休憩、長い休憩をサポート
* **自動進行（Auto-Advance）**: 作業→短い休憩を4サイクル繰り返した後、自動的に長い休憩へ移行する設定が可能
